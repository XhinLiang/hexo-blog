入职新公司已经有一周多，搬家和安置都已经差不多了，是时候制定一下2018年的计划了。

## 程序员技术方面
2018 年，我打算将自己的注意力集中在 Web 后端方面。

新的团队是直接面向用户的，对可用性和吞吐量要求很高，这对于我一个之前写小脚本的菜鸟来说是一个很大的挑战。

#### Code Review
- 提交代码之前对每一行都自己进行 Review
- 提交 Post-Review 之前自己先确认一遍

#### Java
- 熟练使用 `Streams` API
- 熟悉 `Netty`，深入理解其工作原理，并能阅读一部分源代码
- 熟悉使用 `Spring` 和 `Spring MVC`
- 熟悉使用各种手段进行 `Java` 并发编程
- 熟悉 `JVM` 调优，并能使用各种 `Java` 辅助工具，包括但不限于 `jstack` `jps` 等等
- 复习 《深入理解 Java 虚拟机》《Effective Java 3rd》
- 混迹的社区中增加 [并发编程网](http://ifeve.com/)

#### 其他
- 进一步熟悉使用 `VIM` `iTerm2` 和 `ZSH`
- 熟悉 `macOS` 上各种生产力工具的使用，包括但不限于 `Dash` `Alfred` 等等

## 生活方面

- 跟心爱的胖胖领证（这件事情非常重要）
- 坚持健身
- 每周打电话回家
